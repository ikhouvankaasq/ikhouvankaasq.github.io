🎨 AI Challenge: “Dynamic Story-Driven Canvas Generator”
🎯 De Opdracht

Maak één HTML-bestand met JavaScript dat:

Een interactief verhaal of scène genereert op <canvas> gebaseerd op een prompt of input van de gebruiker.

De AI moet automatisch:

Figuren tekenen (karakters, objecten)

Achtergronden en animaties maken

Kleuren, vormen en posities kiezen op basis van een verhaalthema

Gebruiker kan:

Tekst invoeren (“een magische boswandeling”)

Op “Generate Scene” klikken

De AI genereert een unieke visuele interpretatie van die prompt volledig in code, zonder externe libraries.

🔥 Extra Moeilijkheid

De AI moet:

Volledig dynamisch tekenen met canvas API

Minimaal 3 objecten interactief maken (bijv. bewegen bij klikken)

Animatie toevoegen (bv. bladeren vallen, een ster beweegt)

Kleurpalet automatisch kiezen passend bij de mood van de prompt

Eventueel tekstballonnen toevoegen met korte beschrijving van objecten

🏆 Bonus (voor echt topniveau)

Meerdere “scènes” in één canvas (layered system)

Objecten reageren op muis of toetsen (interactiviteit)

Storytelling logica: AI “verhaalt” de scène automatisch (tekst + visuals)

Optimalisatie: canvas werkt vloeiend zonder lag
